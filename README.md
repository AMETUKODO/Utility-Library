# clsUtil

**clsUtil** is a reusable C++ utility library that provides a collection of **static helper functions** designed to simplify common programming tasks, reduce code duplication, and improve code reusability across different projects.

## ✨ Features

- 🎲 Generate random numbers within a specified range.
- 🔤 Generate random characters, words, and strings:
  - Lowercase letters
  - Uppercase letters
  - Numbers
  - Symbols
- 🔑 Generate random keys using a customizable format, for example:
- ABCD-EFGH-IJKL-MNOP
   - 📋 Fill arrays with:
- Random numbers
- Random words
- Generated keys
- 🔄 Swap values for multiple data types, including:
- `int`
- `double`
- `bool`
- `char`
- `string`
- `clsDate`
- 🔀 Shuffle integer and string arrays.
- 🔒 Encrypt and decrypt text using a simple key-based algorithm.
- 📑 Generate tab spacing for better formatted console output.

## 🎯 Purpose

The purpose of **clsUtil** is to provide a centralized collection of reusable utility functions that help developers:

- Write cleaner and more organized code.
- Avoid rewriting common functions.
- Improve maintainability.
- Speed up C++ application development.

## 🛠 Technologies

- **Language:** C++
- **Programming Paradigm:** Object-Oriented Programming (OOP)
- **Design Pattern:** Static Utility Class
